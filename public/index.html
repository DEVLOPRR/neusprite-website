How To Use?

Most of the tasks in NeuSprite can be achieved via keyboard (except drawing), Here is the list of Hotkeys for NeuSprite.

By Default Brush Tool is Selected, you will have to hold down the key to select the tool binded to that key & releasing the key will result in selecting the tool back to Brush.

  • E - Eraser Tool
  • I - Eye Dropper Tool
  • Ctrl + S - Save File
  • Ctrl + O - Open File
  • Ctrl + Mouse Wheel - Zoom In/Out
  • Space + Left Mouse Move - Pan Grid
