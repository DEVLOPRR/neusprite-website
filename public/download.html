How To Build From Source?

1. Getting Repository:

git clone https://github.com/DEVLOPRR/NeuSprite

2. Installing Dependencies

cd NeuSprite/
npm install

3. Building Binaries

npm run build

4. Packaging To 7z Archive For Windows

npm run package:win32

4. Packaging To AppImage For Linux

npm run package:linux

5. (Optional) Getting SHA256 Has For Built Binaries

npm run hash
